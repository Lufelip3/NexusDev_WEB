    <section id="glossario">
    <h2>Glossário Front-End</h2>

    <dl>
        <dt>HTML</dt>
        <dd>HyperText Markup Language — linguagem de marcação
            que estrutura o conteúdo de páginas web.</dd>

        <dt>CSS</dt>
        <dd>Cascading Style Sheets — linguagem que define
            a aparência visual dos elementos HTML.</dd>

        <dt>JavaScript</dt>
        <dd>Linguagem de programação que adiciona
            comportamento interativo às páginas web.</dd>

        <dt>Responsividade</dt>
        <dd>Capacidade de um layout se adaptar a diferentes
            tamanhos de tela sem perder usabilidade.</dd>

        <dt>Semântica</dt>
        <dd>Uso de tags HTML pelo seu significado real,
            não apenas pelo efeito visual que produzem.</dd>
    </dl>
